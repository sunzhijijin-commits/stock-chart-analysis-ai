# 📈 株式チャート分析AIアプリ

機関投資家レベルのマルチタイムフレーム分析を提供するReactアプリケーション

## 🌟 主な機能

- **マルチタイムフレーム分析**: 月足、週足、日足、短期足を統合分析
- **4つのトレードスタイル対応**: 
  - 💎 長期投資（数年単位）
  - 🐢 ポジショントレード（数ヶ月単位）
  - 🌊 スイングトレード（数週間単位）
  - ⚡ デイトレード（日中取引）
- **SMC分析**: スマートマネーコンセプト、オーダーブロック、流動性分析
- **古典的テクニカル分析**: 移動平均線、ダウ理論、出来高分析
- **拡張インジケーター対応**: 一目均衡表、ボリンジャーバンド、MACD、RSIなど
- **AIチャット機能**: 分析結果について質問可能
- **トレード計画書自動生成**: Markdown形式でエクスポート可能
- **SNSシェア機能**: X（Twitter）、LINEで結果を共有

## 🚀 セットアップ

### 必要要件

- Node.js 16以上
- Gemini API キー

### インストール

```bash
# 依存関係のインストール
npm install
```

### APIキーの設定

1. Google AI Studioで[Gemini APIキー](https://ai.google.dev/)を取得
2. プロジェクトルートに `.env` ファイルを作成：

```bash
# .env.exampleをコピー
cp .env.example .env
```

3. `.env` ファイルを開き、取得したAPIキーを設定：

```env
REACT_APP_GEMINI_API_KEY=your_actual_api_key_here
```

⚠️ **重要**: `.env` ファイルは `.gitignore` に含まれており、Gitにコミットされません。

### 開発サーバーの起動

```bash
npm start
```

## 📖 使い方

1. **チャート画像をアップロード**: 複数の時間軸のチャート画像をドラッグ&ドロップまたはクリックでアップロード
2. **トレードスタイルを選択**: 自分のトレードスタイルに合わせて期間を選択
3. **分析を開始**: 「分析を開始（買い）」ボタンをクリック
4. **結果を確認**: 総合評価、SMC分析、トレード戦略などを確認
5. **AIと対話**: チャット機能で追加の質問や詳細を確認

## 🛠️ 技術スタック

- **フロントエンド**: React 18
- **UI**: Tailwind CSS風のカスタムスタイリング
- **アイコン**: Lucide React
- **AI**: Google Gemini 2.5 Flash API
